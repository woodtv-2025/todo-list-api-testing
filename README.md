# Todo List API Testing

## Описание
Полный набор тестов для REST API с использованием Postman и Mockoon

## Что включено:
- ✅ Основные CRUD операции
- ✅ Граничные и негативные тесты  
- ✅ Mock-сервер с условной логикой
- ✅ Автоматические тесты в Postman

## Технологии:
- Postman (коллекции, тесты, переменные)
- Mockoon (mock API, правила роутинга)
- REST API testing
